Correct. Vulkan is behind Python in two places:
Nozzle equation
Python uses the new quadratic radius and analytic slope.
Generated GLSL still uses linear interpolation.

Wall-contact deduplication
Python deduplicates valid boundary contacts by physical wallFlag.
Vulkan deduplicates by boundary marker TargetID.
Two adjacent markers representing the same wall can therefore apply the wall force twice in Vulkan.

Both must be updated before Python and Vulkan are behaviorally aligned.
