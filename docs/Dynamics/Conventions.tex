\section{Conventions}

\subsection{Geo Dynamics Direction}

The Geo restart is governed by \texttt{GeoConventions.tex}.  Its long-term
target is an instantaneous, purely geometric collision model.  Before editing
Geo dynamics code, call out any proposed change that would require persistent
collision memory, stored contact reservoirs, long-lived first-contact velocity,
phase history, or global writable shared state.
