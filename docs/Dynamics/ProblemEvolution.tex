\section{Problem Evolution}

\subsection{Source-Local Area Weighting Breaks Conservation}

\subsubsection{<previous form>}

The unweighted geometric pair impulse was:

\begin{equation}
J_{ij}=q_{ij}A_{ij}\Delta t.
\end{equation}

Because pair overlap area and pair stiffness are symmetric,
\(A_{ij}=A_{ji}\) and \(q_{ij}=q_{ji}\), independently evaluated source and
target impulses have equal scalar magnitude.

\subsubsection{<current form>}

\texttt{WeightedDynamics} applies a source-local overlap-area weight to the raw
impulse:

\begin{equation}
J_{ij}
=
q_{ij}A_{ij}
\left(
\frac{A_{ij}}{\sum_k A_{ik}}
\right)
\Delta t.
\end{equation}

The denominator is the total overlap area for the current source.  When the two
particles have different surrounding contact sets:

\begin{equation}
\sum_k A_{ik}\ne\sum_k A_{jk},
\end{equation}

and therefore:

\begin{equation}
J_{ij}\ne J_{ji}.
\end{equation}

\subsubsection{difference outcome}

No solution.

\subsubsection{<Analysis>}

The source-owned execution structure remains valid: each source reads the same
frame-start snapshot, vector-sums its contact contributions, and writes only
its own velocity.  The conservation failure is in the source-local impulse
magnitude.  Different source contexts calculate different magnitudes for the
same pair contact, so the independently applied impulses are not
equal-and-opposite.

The current \texttt{ThreeParticle} weighted capture ends with:

\begin{verbatim}
vector momentum drift = approximately 0.04899135
kinetic-energy drift  = +0.00090284
internal momentum     = 0
\end{verbatim}

The internal-momentum ledgers drain completely, and accumulated compression
and release impulse balance, but whole-system vector momentum and kinetic
energy do not return to their starting values.  Complete scalar ledger return
therefore does not establish conservation.

Source-local area weights may describe a source allocation, but multiplying
them into the physical pair impulse breaks pair compatibility.  The next
dynamics investigation should remove source-local weighting from the physical
pair impulse and evaluate whether an unweighted pair contribution restores
equal-and-opposite source evaluations.
