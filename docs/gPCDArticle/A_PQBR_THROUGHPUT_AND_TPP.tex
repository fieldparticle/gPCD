\caption[TITLE:PQBR Performance Plots]{(a) Comparison of randomly ordered particle occupancy (red squares)
and sequentially ordered particle occupancy (blue circles). Sequential
ordering consistently produces lower execution times throughout the
evaluated particle range. The green curve shows the execution-time
difference between the two configurations. The approximately linear
increase in this difference indicates that the performance impact of
random memory access grows proportionally with particle count. A linear
fit of the difference curve yields a locality penalty rate of
\PPBQDiffSlope\ ms per million particles.

(b) Log--log throughput as a function of particle count for the total
application (red), compute stage (blue), and graphics stage (black).

saturation, and throughput-limited regimes. Throughput saturates for
$N>\numprint{\APQBRTIMEPERPALLstartval{}}$.

\Description[TITLE:PQBR Performance Plots]{(a) Comparison of randomly ordered particle occupancy (red squares)

\nc{Locality Penalty}, demonstrating the importance of memory locality
despite identical algorithmic complexity.
